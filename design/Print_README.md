# Wearable Fabric — 3D Printed Mesh Fabric
The wearable enclosure for the Human Operator device is made from 3D printed flexible mesh tiles. The mesh layer can be replaced by thin polyester or Nylon which melting point is above 250 °C


## How to Use the Mesh

1. Slice and Print Scale Pattern according to your needs. The file is 5*10cm. The initial layer is 0.2mm and each later is 0.24mm
2. Stop printing at the seocond layer
3. Carelly lay mesh over the printed layers and fix with magnets. 
4. Ensure the tension of the mesh layer is uniform; the mesh does not need to be extremely taut, but it must not have any bulges or unevenness.
5. Continue Printing till finished
   
## Hardware Used

Printed on Bambu Lab P2S and H2S printers.
Material: PLA
Layer height: 0.24mm | No supports required.
